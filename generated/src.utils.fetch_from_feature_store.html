src.utils.fetch_from_feature_store

src.utils.fetch_from_feature_store(from_datetime: str, to_datetime: str | None, feature_refs: List[str], entities: Dict[str, str | List[str]] | None = None) → DataFrame[source]

Fetches historical feature data from Feast feature store within a specified time range and entity constraints.

Parameters:

  • from_datetime (str) – Start datetime for the feature fetch in ISO format (e.g., ‘2023-01-01 00:00:00’).

  • to_datetime (Optional[str]) – End datetime for the feature fetch in ISO format. If None, fetches up to the latest available data.

  • feature_refs (List[str]) – List of feature references in the format ‘feature_view_name:feature_name’.

  • entities (Optional[Dict[str, Union[str, List[str]]]]) – Dictionary of entity filters where: - key: entity column name - value: either a single entity value (str) or list of entity values (List[str]) Defaults to None, which means no entity filtering.

Returns:

DataFrame containing the requested features with columns:

  • event_timestamp: timestamp of the feature values

  • entity columns: as specified in the feature view

  • feature columns: as requested in feature_refs

Return type:

pandas.DataFrame

Example

>>> from src.utils import fetch_from_feature_store
>>> feature_refs = ['raw_ohlcv:close', 'template_features:std_log_return_10_close']
>>> entities = {'exchange': ['KuCoin']}
>>> df = fetch_from_feature_store(
...     from_datetime='2024-12-01 00:00:00',
...     to_datetime='2024-12-10 00:00:00',
...     feature_refs=feature_refs,
...     entities=entities
... )

>>> #to get all feature references in a feature view
>>> from src.config import FEAST_STORE
>>> feature_view = FEAST_STORE.get_feature_view("raw_ohlcv")
>>> feature_refs = [f"{feature_view.name}:{feature.name}" for feature in feature_view.features]
>>> entities = {'exchange': ['KuCoin']}
>>> df = fetch_from_feature_store(
...     from_datetime='2024-12-01 00:00:00',
...     to_datetime='2024-12-10 00:00:00',
...     feature_refs=feature_refs,
...     entities=entities
... )
